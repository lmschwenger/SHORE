About {{ app_name }}
{{ app_description }}

This application uses the STAC API (SpatioTemporal Asset Catalog) to search for and retrieve Sentinel satellite imagery from the Copernicus Data Space Ecosystem.

Features:
  • Draw areas directly on the map
  • Search for Sentinel images by date range
  • Filter images by cloud coverage
  • View image previews and metadata
  • Access to all available bands and assets

Developed using Flask, Leaflet, and the Copernicus STAC API.
